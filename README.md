# 🎬 Front-end Netflix Clone

Projeto de interface inspirado em plataformas de streaming, desenvolvido durante a Imersão Front-End da Alura.

## 📌 Funcionalidades

- Gerenciador de perfis com salvamento automático (localStorage)
- Tema escuro/claro
- Interface responsiva para diferentes dispositivos

## 🛠️ Tecnologias utilizadas

- HTML
- CSS
- JavaScript

## 📚 Créditos

O catálogo utilizado foi disponibilizado pelo professor Guilherme Lima durante a imersão:

https://github.com/guilhermeonrails/netflix-plataform

Foram realizadas adaptações para integração com o sistema de perfis e organização do projeto.

## 💡 Aprendizados

Durante o desenvolvimento deste projeto, aprendi:

- Estruturação de páginas com HTML
- Estilização e responsividade com CSS
- Manipulação de dados e lógica com JavaScript
- Integração entre front-end e armazenamento local (localStorage)
- Organização de código e separação de responsabilidades (HTML, CSS e JS)

## 🚀 Próximos passos

Este projeto serviu como base para ideias futuras, com o objetivo de evoluir para uma aplicação mais completa de organização de conteúdos e projetos pessoais.
